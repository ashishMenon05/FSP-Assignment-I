# Online Examination System Demo

## Overview
This project is a **client-side demo of an Online Examination System** built using **HTML, CSS, and JavaScript**. It demonstrates the key features of an online exam platform, focusing on **client-side responsiveness**, including:

- Real-time countdown timer
- Multiple-choice questions (MCQs)
- Instant answer selection highlighting
- Progress bar showing number of questions answered
- Submit button with alert summarizing selected answers

> This project is intended as a **college assignment demo** to showcase the functionality of a responsive online examination interface. All operations are handled on the **client side**, with no server/backend required.

---

## Features
- **5 MCQs** with 4 options each
- **Countdown Timer** starting from 5 minutes
- **Answer Highlighting** when selected
- **Progress Bar** to track answered questions
- **Submit Exam** button shows selected answers in an alert
- Clean, professional UI using CSS
